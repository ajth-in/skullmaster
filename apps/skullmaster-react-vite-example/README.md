# Neo Brutalist UI Showcase

A showcase of common web UI components in a neo brutalist design style, built with React + TypeScript + Vite.

## Features

- **Loading Context** — Global `isLoading` state with a toggle button in the top-left corner.
- **Hero Section** — Bold, oversized typography with a brutalist color palette.
- **User Profile Card** — Avatar, stats, and action button in a chunky card layout.
- **Common Components** — Buttons, form controls, badges, alerts, progress bars, and mini-cards.
- **Neo Brutalist Design** — Thick black borders, offset box shadows, high contrast, raw aesthetic.
